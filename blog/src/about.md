## About SuiDev Labs

SuiDev Labs is a technical blog focused on the **Sui blockchain ecosystem**, decentralized finance (DeFi), and the **Move programming language**. We provide tutorials, deep dives, and ecosystem updates for developers building on Sui.

### What We Cover

- **Sui Development** - Smart contract development using Move, Sui SDK integration, and best practices
- **DeFi Protocols** - Analysis of AMMs, lending protocols, and yield strategies on Sui
- **Move Language** - Language features, patterns, and optimization techniques
- **Ecosystem Updates** - News, protocol launches, and developer tooling

### For Developers, By Developers

Our content is written by experienced blockchain developers who are actively building on Sui. We focus on practical, hands-on tutorials that you can use in your own projects.

### Open Source

This blog is built with [Eleventy](https://www.11ty.dev/) and hosted on GitHub Pages. The source code is available at [github.com/suidev-labs/suidev-labs.github.io](https://github.com/suidev-labs/suidev-labs.github.io).
